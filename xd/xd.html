Kawsay Roast

Descripción

Kawsay Roast es una página web desarrollada como proyecto académico con el propósito de aplicar conocimientos de diseño web, organización de contenido y experiencia de usuario.

Objetivos del proyecto

Diseño moderno

Crear una interfaz atractiva y profecional que refleje una imagen limpia y actual

Experiencia de usuario

Facicilitar una navegación intiutiva y accesible para todos los vivitantes del sitio

Identidad peruana

Representar elementos culturales y productos del Perú mediante una plataforma digital.
